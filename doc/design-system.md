# Habit Tracker App — Design System
Version: 1.0  
Platform: iOS + Android  
Style: Minimal / Soft Modern / Productivity Wellness

---

# 1. Product Vision

The app should feel:

- Calm
- Motivating
- Clean
- Lightweight
- Encouraging
- Non-overwhelming

Design inspiration:

- Apple Health
- Headspace
- Notion Mobile
- Calm
- Duolingo streak system

Core emotional goal:

> “Small progress every day.”

---

# 2. Design Principles

## 2.1 Simplicity First

- Avoid clutter
- One primary action per screen
- Large whitespace
- Minimal borders

## 2.2 Positive Reinforcement

- Celebrate streaks
- Encourage consistency
- Avoid negative/error-heavy UX

## 2.3 Soft UI

- Rounded corners
- Soft shadows
- Light gradients
- Friendly illustrations/icons

## 2.4 Accessibility

- Minimum touch target: 44x44
- High contrast text
- Dynamic font scaling
- Color should never be the only indicator
